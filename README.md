A simple FAQ Bot for Delta Chat groups

## Install

```sh
pip install -U faqbot
```

## Usage

Configure the bot:

```sh
faqbot init bot@example.com PASSWORD
```

Start the bot:

```sh
faqbot serve
```

Run `faqbot --help` to see all available options.
